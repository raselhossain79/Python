## 01. How Python Works

Python is an **interpreted** language. Code runs line by line — no compilation step needed.

```
You write code → Python interpreter reads it → Executes immediately
```

### Python 2 vs Python 3
Always use **Python 3**. Python 2 is dead (EOL 2020).

```bash
python3 --version       # Check version
python3 script.py       # Run a script
python3                 # Open interactive shell (REPL)
```

### Interactive Shell (REPL)
Great for testing small pieces of code quickly:
```python
>>> 2 + 2
4
>>> name = "Rasel"
>>> print(name)
Rasel
>>> exit()
```

### Your First Script
```python
# hello.py
print("Hello, World!")
print("Python version check complete")
```

```bash
python3 hello.py
```

### Code Style (PEP 8 — Python's official style guide)
- Indent with **4 spaces** (not tabs)
- Variable names: `snake_case` (e.g. `target_ip`, `open_ports`)
- Constants: `UPPER_CASE` (e.g. `MAX_THREADS = 50`)
- Class names: `PascalCase` (e.g. `PortScanner`)
- Lines should be under 79 characters
- Comments start with `#`

```python
# This is a single-line comment

"""
This is a
multi-line comment (docstring)
"""
```

## 08. Dictionaries

Dictionaries store **key-value pairs**. Like a real dictionary — look up a key, get a value. Extremely useful for structured data.

## 09. Sets

Sets are **unordered collections with no duplicates**. Great for deduplication.

```python
# Create
unique_ips = {"192.168.1.1", "10.0.0.1", "192.168.1.1"}
print(unique_ips)   # {'192.168.1.1', '10.0.0.1'} — duplicate removed

# From list (deduplication)
ips_with_duplicates = ["192.168.1.1", "10.0.0.1", "192.168.1.1", "10.0.0.2"]
unique = set(ips_with_duplicates)
# Back to list:
clean_list = list(unique)

# Operations
s1 = {1, 2, 3, 4}
s2 = {3, 4, 5, 6}

s1 | s2     # Union: {1, 2, 3, 4, 5, 6}
s1 & s2     # Intersection: {3, 4}
s1 - s2     # Difference: {1, 2}
s1 ^ s2     # Symmetric difference: {1, 2, 5, 6}

# Modify
unique.add("172.16.0.1")
unique.remove("10.0.0.1")   # Error if not found
unique.discard("10.0.0.1")  # Safe — no error if not found

# Check
"192.168.1.1" in unique     # True/False
```

## 12. List Comprehensions

A compact way to create lists. Very Python-specific — used everywhere.

```python
# Basic syntax: [expression for item in iterable if condition]

# Without comprehension:
squares = []
for i in range(10):
    squares.append(i ** 2)

# With comprehension:
squares = [i ** 2 for i in range(10)]

# With condition:
even_squares = [i ** 2 for i in range(10) if i % 2 == 0]

# Real examples:
ports = [22, 80, 443, 8080, 8443]
high_ports = [p for p in ports if p > 1024]           # [8080, 8443]
port_strings = [str(p) for p in ports]                 # ["22", "80", ...]
open_ports = [p for p in ports if is_open(p)]

# Load and clean file:
with open("targets.txt") as f:
    targets = [line.strip() for line in f if line.strip()]

# Flatten nested list:
nested = [[1, 2], [3, 4], [5, 6]]
flat = [x for sublist in nested for x in sublist]  # [1,2,3,4,5,6]
```

### Dictionary & Set Comprehensions

```python
# Dict comprehension
```

### Docstrings
Always document your functions:
```python
def scan_port(host, port, timeout=1):
    """
    Check if a TCP port is open on a host.

    Args:
        host (str): Target IP or hostname
        port (int): Port number to check
        timeout (int): Connection timeout in seconds

    Returns:
        bool: True if port is open, False otherwise
    """
    import socket
    try:
        s = socket.socket()
        s.settimeout(timeout)
        s.connect((host, port))
        s.close()
        return True
    except:
        return False

help(scan_port)     # Shows the docstring
```

### Scope — Local vs Global
```python
target = "global"       # Global variable

def scan():
    target = "local"    # Local — different from global
    print(target)       # "local"

scan()
print(target)           # "global" — unchanged

# Modify global inside function:
count = 0
def increment():
    global count
    count += 1          # Now modifies global

# Better approach — avoid global, use return:
def increment(count):
    return count + 1

count = increment(count)
```

## 14. Lambda, map, filter

### Lambda — Anonymous one-line function
```python
# Normal function:
def square(x):
    return x ** 2

# Lambda:
square = lambda x: x ** 2
print(square(5))    # 25

# With two parameters:
add = lambda x, y: x + y
print(add(3, 4))    # 7

# Most common use — as key in sort:
ports = [(80, "http"), (22, "ssh"), (443, "https")]
ports.sort(key=lambda x: x[0])    # Sort by port number
```

### map — Apply function to every item
```python
ports = ["22", "80", "443"]
port_ints = list(map(int, ports))       # ["22","80","443"] → [22, 80, 443]

names = ["rasel", "admin", "root"]
upper_names = list(map(str.upper, names))

# Same as comprehension (prefer comprehension):
port_ints = [int(p) for p in ports]
```

### filter — Keep only items that match condition
```python
ports = [22, 80, 443, 8080, 8443, 21]
high_ports = list(filter(lambda p: p > 1024, ports))   # [8080, 8443]

# Same as comprehension:
high_ports = [p for p in ports if p > 1024]
```

## 15. File I/O

### Reading Files
```python
# Method 1 — with statement (ALWAYS use this — auto-closes file)
with open("targets.txt", "r") as f:
    content = f.read()          # Read entire file as string

with open("targets.txt", "r") as f:
    lines = f.readlines()       # Read all lines into list (with \n)

with open("targets.txt", "r") as f:
    for line in f:              # Read line by line (memory efficient)
        target = line.strip()
        print(target)

# Clean way — strip and skip empty lines:
with open("targets.txt") as f:
    targets = [line.strip() for line in f if line.strip()]
```

### Writing Files
```python
# Write (overwrite)
with open("results.txt", "w") as f:
    f.write("Scan Results\n")
    f.write(f"Target: 192.168.1.1\n")

# Append
with open("results.txt", "a") as f:
    f.write("New result\n")

# Write multiple lines
results = ["192.168.1.1 - open", "10.0.0.1 - closed"]
with open("results.txt", "w") as f:
    f.writelines(line + "\n" for line in results)
    # Or:
    f.write("\n".join(results))
```

### File Modes
| Mode | Meaning |
|---|---|
| `"r"` | Read (default) |
| `"w"` | Write (overwrite) |
| `"a"` | Append |
| `"rb"` | Read binary |
| `"wb"` | Write binary |
| `"r+"` | Read and write |

### File Operations with os
```python
import os

os.path.exists("file.txt")      # True/False
os.path.isfile("file.txt")      # Is it a file?
os.path.isdir("directory/")     # Is it a directory?
os.path.getsize("file.txt")     # File size in bytes
```

## 20. OOP Basics

Object-Oriented Programming. You need to understand this to properly use libraries like `requests`, `socket`, `paramiko`.

### Classes & Objects
A **class** is a blueprint. An **object** is an instance of that blueprint.

```python
class Scanner:
    # Class variable (shared by all instances)
    tool_name = "PyScanner"

    # __init__ = constructor (runs when object is created)
    def __init__(self, target, timeout=5):
        # Instance variables (unique to each object)
        self.target = target
        self.timeout = timeout
        self.open_ports = []

    # Method (function inside class)
    def scan_port(self, port):
        import socket
        try:
            s = socket.socket()
            s.settimeout(self.timeout)
            s.connect((self.target, port))
            s.close()
            self.open_ports.append(port)
            return True
        except:
            return False

    def get_results(self):
        return {
            "target": self.target,
            "open_ports": self.open_ports
        }

    def __str__(self):
        # What prints when you print(object)
        return f"Scanner({self.target})"


# Create objects
scanner1 = Scanner("192.168.1.1")
scanner2 = Scanner("10.0.0.1", timeout=3)

# Use methods
scanner1.scan_port(80)
scanner1.scan_port(443)
print(scanner1.get_results())
print(scanner1)     # Calls __str__
```

### Inheritance
```python
class WebScanner(Scanner):      # Inherits from Scanner
    def __init__(self, target):
        super().__init__(target)    # Call parent __init__
        self.web_ports = [80, 443, 8080, 8443]

    def scan_web(self):
        for port in self.web_ports:
            self.scan_port(port)
        return self.open_ports

web = WebScanner("192.168.1.1")
web.scan_web()
```

## 21. Regular Expressions

## 23. Generators & Iterators

Generators produce values **one at a time** — memory efficient for large data.

```python
# Normal function — creates entire list in memory
def generate_ips(network):
    ips = []
    for i in range(1, 255):
        ips.append(f"{network}.{i}")
    return ips    # All 254 IPs in memory at once

# Generator function — produces one at a time
def generate_ips(network):
    for i in range(1, 255):
        yield f"{network}.{i}"    # yield instead of return

# Use it:
for ip in generate_ips("192.168.1"):
    print(ip)   # Only one IP in memory at a time

# Generator expression (like list comprehension but lazy)
ips = (f"192.168.1.{i}" for i in range(1, 255))
```
